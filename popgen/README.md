# Population genomics analysis
Code and data for population genomics analysis.

- **admixture/**  
  Results of ADMIXTURE analysis.  
- **msmc/**  
  Results of MSMC2 analysis.  
- **pca/**  
  Results of PCA.  
- **Dloop_MK.fst**  
  Multiple sequence alignment of mtDNA D-loop sequences in FASTA format.  
- **Dloop_MK_PopART.nex**  
  Nexus file for haplotype network construction in PopART.
- **MK28.txt**  
  List of 28 sample IDs (mushifugu and komonfugu) used in analyses.  
- **MK28_msmc.ipynb**  
  Jupyter notebook for MSMC analysis.  
- **MK28_pca_admixture.ipynb**  
  Jupyter notebook for PCA and ADMIXTURE analysis.
