# puffer-popgen

[![bioRxiv](https://img.shields.io/static/v1?label=bioRxiv&message=10.1101/2025.11.20.687103&color=bd271e)](https://doi.org/10.1101/2025.11.20.687103)

Custom code and data for the analysis of pufferfish population structure and demographic history.

**"Peculiar Pigment Pattern and Population Profile of a Poisonous Pufferfish"**  
Seita Miyazawa *et al.* (2025)

- **patterns**  
Code and data for pigment pattern quantification.
- **popgen**  
Code and workflows for population genomics analysis.
- **puffer_MK.csv**  
Metadata file for samples used in the analyses (mushifugu and komonfugu).
