# Pattern quantification
Code and data for quantifying pufferfish pigment patterns.

- **komonfugu/**  
  Directory containing binarized images of komonfugu pigment patterns.  
- **mushifugu/**  
  Directory containing binarized images of mushifugu pigment patterns.  
- **quant_CV4.py**  
  Custom module used for pattern quantification.  
- **quant_MK.ipynb**  
  Jupyter notebook for pattern quantification analysis.  
- **uni_p.csv**  
  Data from RD model simulation.  
  (see Miyazawa 2020 *Science Advances*: [DOI: 10.1126/sciadv.abb9107](https://www.science.org/doi/10.1126/sciadv.abb9107))
